# Security and private-data boundary

This repository contains a static Markdown/HTML checklist and release tooling.
It does not accept credentials, execute third-party repositories, or provide a
hosted application.

Do not report secrets, private code, customer data, payment information, wallet
keys, or vulnerability details in a public issue. For a private concern, email
`zach@zachwright.xyz` with the subject `Bounty Red-Flag Card private report`.

The card is a triage aid, not a security assessment. A low flag count does not
establish that a repository, bounty, payer, or workflow is secure.
